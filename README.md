# honoybundemo

This project was created using `bun init` in bun v1.2.13. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.

When cloning repo install dependencies with

```bash
bun install
```

To run: (no needed)

```bash
bun run index.ts
```

In command (no needed)

```bash
bun --watch run index.ts
```

//// run server

```bash
bun run dev
```
